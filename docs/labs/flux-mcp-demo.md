# Flux MCP Server Demo

AI-assisted GitOps debugging with natural language.

!!! info "This is a live demo, not a hands-on lab"
    Steve will demonstrate the Flux MCP Server on a live cluster. Watch how natural language queries replace manual kubectl debugging.

## What You'll See

- **Root cause analysis**: "Why did the HelmRelease fail?" and getting an actual answer
- **Cluster state queries**: "Show me what's not reconciling" across all resources
- **Cross-cluster comparison**: Compare configurations between environments
- **Dependency visualisation**: See how Flux resources relate to each other

## About the Flux MCP Server

The Flux MCP Server is a new component of the Flux Operator project that connects AI assistants directly to your Kubernetes clusters. It implements the Model Context Protocol (MCP), enabling AI tools like Claude, Cursor, and GitHub Copilot to interact with your GitOps pipelines.

It launched in 2025. Most teams haven't seen it yet.
